Compile GPlates on Fedora Linux

Posted: 2023-06-24
Updated: 2024-07-25
Tags: guide, linux, out of date

Note: this guide is out of date. GPlates has a new stable version (2.4.0) and the download link for the old source code doesn’t work anymore. GPlates is a free software program for modeling the tectonic history of a planet. It was designed with a scientific audience in mind, but it has also gained popularity among the worldbuilding community. GPlates has an Ubuntu package and both source- and binary-based Arch User Repository packages, but no package for Fedora Linux. One of the options for running GPlates on Fedora is to compile GPlates from source; this guide will show you how to do so.
