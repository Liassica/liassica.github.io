How to compile GPlates on Fedora Linux

Posted: 2023-05-06
Updated: 2024-01-06
Tags: guide, out of date

Note: this guide is out of date. GPlates has a new stable version (2.4.0) and the download link for the old source code doesn’t work anymore. GPlates is a free software program for modeling the tectonic history of a planet. It was designed with a scientific audience in mind, but it has also gained popularity among the worldbuilding community. GPlates has an Ubuntu package and both source- and binary-based AUR packages, but no package for Fedora. This guide will show you how to compile GPlates from source.
