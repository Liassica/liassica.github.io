yt-dlp guide

C/U/ID: 2023-04-21/2023-04-24/G002

1. What is yt-dlp? Why should I use it?

yt-dlp is a command-line tool for downloading videos and audio from YouTube and other sites. It is a maintained fork of youtube-dl, with new features and improvements.

The benefit of using yt-dlp over a random downloader site is not only the greater control over the output, but it's also a lot safer to use. You don't have to worry about downloading malware with yt-dlp.

2. Installing yt-dlp

Windows

winget, chocolatey, or scoop:

winget install yt-dlp

choco install yt-dlp

scoop install yt-dlp

Linux

apt (Debian/Ubuntu):

sudo add-apt-repository ppa:tomtomtom/yt-dlp
sudo apt update
sudo apt install yt-dlp

dnf (Fedora):

sudo dnf install yt-dlp

pacman (Arch):

sudo pacman -Syu yt-dlp

Other Linux package managers

See the installation page of the yt-dlp wiki.

MacOS

brew:

brew install yt-dlp

3. Simple usage

If you downloaded yt-dlp with a package manager, simply open a terminal to whatever folder you want to download into and type yt-dlp [URL].

If you instead downloaded the binary manually, open a terminal to the folder you put the binary into and type ./yt-dlp [URL]. Alternatively, you can add the folder to your system's PATH, which will allow you to use it anywhere like with package-manager-installed yt-dlp. See appendices for doing so on Windows and Linux/MacOS.

4. Installing FFmpeg

Many of yt-dlp's advanced options require FFmpeg to be installed as well. FFmpeg can be used for a variety of audio-related processing on its own, but it can be used with yt-dlp to allow the download and remuxing of audio in different formats.

FFmpeg can be installed much the same way as yt-dlp, via command line or manually.

Windows

winget or chocolatey, or scoop:

winget install ffmpeg

choco install ffmpeg

scoop install ffmpeg

Linux

apt (Debian/Ubuntu):

sudo apt update
sudo apt install ffmpeg

dnf (Fedora):

First enable RPM Fusion repositories (this is one long command):

sudo dnf install https://mirrors.rpmfusion.org/free/fedora/rpmfusion-free-release-$(rpm -E %fedora).noarch.rpm https://mirrors.rpmfusion.org/nonfree/fedora/rpmfusion-nonfree-release-$(rpm -E %fedora).noarch.rpm

Then run: sudo dnf install ffmpeg

Other Linux distros:

Download manually.

5. Advanced usage

yt-dlp allows many options to be used in order to more finely direct its output. The standard syntax is yt-dlp [OPTIONS] [URL]. The full list of options can be found on the README for yt-dlp.

Common options include:

  • -h - shows the help text
  • -f - change the output format. Sometimes -f won't work when trying to convert audio; try --audio-format in that case.
    • yt-dlp -f mp4 [URL] will download the video as mp4
  • -x - download audio only
    • yt-dlp --audio-format mp3 -x [URL] will download audio as mp3
  • -o - change the output template (file name)
    • yt-dlp -f mkv -o "video2.%(ext)s" [URL] will download the video as video2.mkv. %(ext)s is Python printf syntax for a variable (%) called ext (the extension, in this case mkv) as a string (s). Use this over hardcoding the extension (e.g. -o "video2.mkv"). Read the full output template documentation here.
