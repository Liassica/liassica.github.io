Arkenfox (hardened Firefox) Guide

C/U/ID: 2023-04-10/2023-05-02/G001

1. What is Arkenfox? Why should I use it?

Arkenfox is a user.js (configuration) file for desktop Firefox designed to improve privacy and security while not going so overboard as to make Firefox unusable. It's best used when you want better security and privacy than a stock Firefox installation or Chromium browser, but don't want to deal with Tor browser's slowness and lack of customization.[a] However, Arkenfox is geared towards advanced users. You will have to be prepared to fix things when they break. For most people, a simpler guide such as this one by Geck will do just fine. Read through this page to determine if Arkenfox is right for you.

2. Installing Arkenfox

To install Arkenfox, first install Firefox:

  1. Go to this link. The download should begin automatically.
  2. Once the installer is done downloading, open it and Firefox will install on your computer. Open Firefox when done, if it doesn't start automatically.
  3. Firefox will show a pop-up about importing bookmarks and other setup things. Click through these, they won't matter because you'll be making a new profile.

Next, we are going to make a new Firefox profile.

  1. In the address bar, type 'about:profiles' to open the profiles page.
  2. You'll see some default profiles, ignore these. Click 'Create a New Profile'.
  3. In the profile creation wizard that pops up, hit next. Name the profile however you want (e.g. 'Arkenfox Profile') and hit next.
  4. The profile creation wizard will close. Scroll down to the new profile and hit 'Launch profile in new browser'. This will initialize the new profile. Firefox may pop-up with the same set-up box, and you can either just ignore it again or look through it.

After that, we are going to install the Arkenfox user.js file in the profile directory.

  1. Go to 'about:support'. In the list under 'Application Basics', find 'Profile Folder' and hit the 'Open Folder' button. Keep the folder that opens open during the next steps.
  2. Go to the Arkenfox user.js download page and download either 'Source code (zip)' if you're on Windows or 'Source code (tar.gz)' if you're on Linux.
  3. Extract the downloaded file. You should see a variety of files. If you're on Windows, copy prefsCleaner.bat, updater.bat, and user.js to the profile folder you opened earlier. If you're on Linux, copy prefsCleaner.sh, updater.sh, and user.js instead.
  4. Now, time for the optional user-overrides.js. A user-overrides.js file is used by the Arkenfox updater to apply user-specific settings on top of the Arkenfox user.js. Do not manually edit user.js. I've created a few very simple user-overrides.js files that you can read about and download here. As you get more familiar with Firefox, you can make your own user-overrides.js file. To use one of the user-overrides.js files I made, simply download the .zip file with the one you want, extract it, and copy user-overrides.js into your profile folder.
  5. Once you've copied all of the files over, exit Firefox (control + shift + q).

Next is running the Arkenfox updater. This will append your user-overrides.js file to your user.js file. It will also ensure that your Arkenfox is up to date. As a general rule, run the updater after every Firefox update.

  1. Run updater.bat or updater.sh from your profile folder.
  2. Type 'S' to start the updater. It should only take a few seconds.
  3. If you get a prompt asking to run the prefsCleaner, type 'Y'. This will open the prefsCleaner, which makes sure that old preferences get removed and new preferences are properly updated.
  4. Type 'S' to run the prefsCleaner. If you get a prompt saying "Firefox is still running", make sure to exit Firefox, not just close it. Once you've exited Firefox, hit any key to continue.
  5. PrefsCleaner should close, and then you're done! Your newly hardened Firefox should be ready to use next time you launch it.

3. Closing thoughts

Further customization of Firefox is possible with extensions. The one extension I explicitely recommend is uBlock Origin. It is a powerful (even out of the box) content blocker that blocks ads, malicious domains and scripts, trackers, pop-ups, etc. Using it will vastly improve both privacy and security (as well as performance). From there, extensions are up to you, though do keep in mind that the more extensions you have, the more vulnerable your browser is. Read through the extensions section of the Arkenfox wiki, especially the "don't bother" section for more on extensions.

If you encounter issues while using Arkenfox, check the common overrides page of the Arkenfox wiki. Chances are changing one of these will fix your issue.

Enjoy your faster, more secure, and more private Firefox!

  1. ^ Even the most hardened Firefox setup will not be as private as Tor. However, as noted above, Tor is very slow, due to having to make connections through the TOR network.
