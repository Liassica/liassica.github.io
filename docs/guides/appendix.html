Modifying the PATH on Windows

C/U/ID: 2023-04-21/2023-04-21/A001

To modify the PATH on Windows:

  • Open the Windows search and search for "path"
  • Click on "Edit the system environment variables"
  • This will open a system properties dialogue box; click "Environment Variables" in the bottom right
  • In either the user variables or system variables sections of the next dialogue box, find "Path", select it, and hit edit
  • In the subsequent box, click "New" and then "Browse" and then navigate to the folder you want to add to the PATH
  • Hit "OK" on each box until you close the system properties

Modifying the PATH on Linux/MacOS

C/U/ID: 2023-04-21/2023-04-21/A002

To modify the PATH on Linux or MacOS:

  • Open a terminal and navigate to your home directory if you're not there already: cd ~
  • Open the .bashrc or .bash_profile file (or your shell's eqivalent, such as .zshrc or .zsh_profile for z shell) in nano: nano .bashrc
  • Add this line: export PATH=$PATH:/path/to/directory with the path(s) to the desired directory(s), colon-separated
    • For example, to add the ~/Programs/syncthing directory to the path, the line should be export PATH=$PATH:/home/[USER]/Programs/syncthing
    • To add the ~/Programs/yt-dlp and ~/Tools/FFmpeg directories to the path, the line should be export PATH=$PATH:/home/[USER]/Programs/yt-dlp:/home/[USER]/Tools/FFmpeg
  • Once your desired PATH is specified, hit ctrl+O to save the file, press enter to use the default file name, and then press ctrl+X to exit nano
  • Restart the terminal or type source ~/.bashrc (or whichever file you edited)
